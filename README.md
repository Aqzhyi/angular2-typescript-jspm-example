# angular2-typescript-jspm-example

> An trial for using angular2, typescript and jspm to build web application.

## Start dev

> This trial working with Chrome 46

```sh
npm install
tsd install
jspm install
gulp build
http-server
open http://0.0.0.0:8080/
```
